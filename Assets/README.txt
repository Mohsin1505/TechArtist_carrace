Technical Artist Assignment – Stylized Runner

Overview

Stylized endless runner built in Unity (URP) focusing on visual quality, game feel, and performance for low-end devices.

Visuals

Used a clean color palette with high vibrance similar to the reference style. Ensured strong contrast between the car and environment for readability.

 Camera

Implemented a custom script for smooth X-axis follow using interpolation, with slight delay and tilt to improve responsiveness.

VFX

added the swipe lane particle trigger in car controller Added particle effects for lane switching (dust) and motion feedback using optimized particle systems.

Optimization

Reduced batches and shadow casters for better performance on low-end devices.
Used minimal materials, optimized textures, and repeated objects to reduce rendering cost.

Assets

Environment props are based on the SimplePoly City asset pack.
